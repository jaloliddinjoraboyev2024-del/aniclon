========================================
 ANICLON - Anime sayti
 Joylash ko'rsatmasi
========================================

1. GitHub'ga joylash:
   -------------------
   1. GitHub.com saytiga kiring va hisobingizga kiring
   2. "New repository" tugmasini bosing
   3. Repository nomi: aniclon (yoki xohlagan nom)
   4. Public qilib yarating
   5. Barcha fayllarni yuklang (drag and drop yoki git orqali)
   
2. Netlify'ga ulash:
   -----------------
   1. Netlify.com saytiga kiring (GitHub orqali kiring)
   2. "Add new site" → "Import an existing project"
   3. GitHub ni tanlang va aniclon repository ni tanlang
   4. Deploy tugmasini bosing
   5. Sayt tayyor! Domen: aniclon.netlify.app
   
3. Netlify CMS (to'liq admin panel):
   ---------------------------------
   1. Netlify dashboard'da saytingizni oching
   2. "Identity" bo'limiga o'ting
   3. "Enable Identity" ni yoqing
   4. "Git Gateway" ni yoqing
   5. Endi siz saytingizga /admin/ orqali kirib, animelarni boshqara olasiz

4. Animelarni qo'shish:
   --------------------
   - Hozircha localStorage orqali: admin/index.html
   - Netlify CMS sozlangandan keyin: sayt.uz/admin/
